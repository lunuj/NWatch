# 自动亮度

#### 器件：光敏传感器

#### 接口：VCC接正，GND接负，AO接ADC通道1即P12口

#### 函数：OLED

#### 输出：0-4100模拟量（由亮变暗）

# 闹钟

#### 器件：蜂鸣器

#### 接口：VCC接正，GND接负， I/O接PA6

#### 函数：Alarm

#### 输入：低电平有效

# 检测物体亮屏

#### 器件：反射传感器

#### 接口：VCC接正，GND接负，DO接P13

#### 函数：LightSensor

#### 输出：检测到物体输出高电平无物体为低电平

# 温度检测

#### 器件：DHT11

#### 接口：S接PB1

#### 函数：DHT11

#### 输出：u8 buffer[5]

- #### buffer[0] 为温度高位

- #### buffer[1] 为温度低位

- #### buffer[2] 为湿度高位

- #### buffer[3] 为湿度低位

- #### buffer[4] 为校验位

# 翻转亮屏

#### 器件：MPU6050

#### 接口：VCC接正，GND接负，SCL接PB10，SDA接PB11

#### 函数：Kalmen

#### 输出：k_pitch，k_roll

# 数据传输

8位亮度 16位温度 16位湿度 8位roll



串口 RX Pin10 TX Pin9



esp32 RX pin16 TX Pin17
